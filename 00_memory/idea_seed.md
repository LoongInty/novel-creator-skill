# 剧情种子（模糊输入转结构化）

## 一句话脑洞
战后废墟中，一个11岁孤儿凭借超高速大脑暗中掌握了人类所有技术，以科技解析修炼，建立全球势力寻找失踪父母，最终发现一切指向一个古老的空间裂隙——而他的一生，从出生起就已经被裂隙绑定了。

## 题材与目标读者
- 题材：都市异能 + 科技修真 + 悬疑寻亲
- 目标读者：18-40岁
- 写作风格：轻松幽默，痞气主角，重点描写不省
- 核心禁区：无禁区，什么都能写
- 自动化等级：全自动（完全托管）
- 目标规模：300章，每章内容长度不限

## 主角（起点状态）
- 姓名：待定
- 年龄：故事开始11岁
- 出身：普通研究员家庭，父母战前失踪 → 战后孤儿院
- 性格底色：痞气不羁型，玩世不恭，嘴贱但重情义
- 金手指：纯粹智力侧
  - 大脑超高速运转，别人思考几十年的东西在他脑中只需几毫秒
  - 能洞察万物运行原理，看一遍就知道本质
  - 五六岁时暗中攻破全球数据库，获取了所有技术和秘密
  - 推演出超越本时代几千年的理论：能量不守恒、物质不守恒等
  - 根据修炼信息自创最适合自己的功法
  - 针对自身做了身体改造（洗髓、体质改造等）
- 修炼路线：科技解析修炼——用数学和物理模型重构修炼体系
- 起点处境：战后中国某城市孤儿院，伪装成普通孤儿
- 弱点：寻找父母阶段遭遇各种阻力，最开始以为是运气不好，后期发现是各方势力在隐藏/阻止

## 核心冲突
1. **主线冲突**：寻找失踪父母 → 卷入世界背后的真相（空间裂隙）
2. **势力博弈**：深境实验室（看守封印）↔ 彼岸（天魔组织，利用主角）↔ 净世会（激进派，刺杀主角）
3. **终极反转**：一直帮助主角的组织（彼岸）其实是逃逸天魔，目的是利用主角打开裂隙

## 世界观
- 时间：第三次世界大战后（代理人战争，非洲主战场），停战协议期
- 技术状态：先进技术大面积丢失（实验室、数据机房被毁）
- 黑色人种在世界大战中消失（基因武器结果）
- 修炼体系：自古传承，起源未知；经典九境（炼气→筑基→金丹→元婴→化神→合体→渡劫→大乘→飞升）
- 全球存在其他地域的多元修炼体系和不同等级划分
- 修炼者与普通人关系：半公开，世人几乎没见过真正的修炼者
- 故事发生地：中国某战后重建城市 → 非洲 → 复兴学院

## 主要势力
- **深境实验室**：原"门扉计划"，研究空间裂隙的机构，看守裂隙封印
- **彼岸**：天魔组织，从裂隙逃逸的存在，表面帮助主角实为利用
- **净世会**：激进派，主张清除所有裂隙相关者以彻底杜绝威胁
- **复兴学院**：战后中国成立的特殊人才培养机构，主角回国后进入

## 核心人物
- **主角**：11岁孤儿，拥有超高速大脑，暗中掌握人类全知识
- **父母**：研究员，"门扉计划"核心科学家，发现空间裂隙后自愿进入维持封印
- **苏晚**：零番队001号，孤儿院同伴，比主角大两岁，密码破译天赋
- **老哑巴**：孤儿院扫地老人，实际是门扉计划第一代研究员，编号001
- **"镜"**：彼岸组织联络人，女性，约20多岁，引导主角打开裂隙

## 核心组织
- **零番队**：49人精英小队，各有所长，绝对忠诚。最终幸存25人，散落世界各地等待重新集结

## 预期结局
主角费尽千辛万苦在深境实验室地下20层见到父母，放入吊坠（裂隙钥匙的一半）解除封印 → 空间裂隙大开 → 天魔涌出 → 真相揭露（彼岸=天魔，吊坠=钥匙，他被利用了） → 主角下令清除彼岸 → 零番队防守战（24人牺牲） → 主角耗尽一切击溃裂隙 → 消失，只留传说。尾章：25名幸存者散落各地，等待暗号重新集结；某个雨夜，最年轻的成员在东京街头看到了一个熟悉的身影，捡起一个被捏成星形的竹签，里面传来微弱的、只有他能听见的心跳。
