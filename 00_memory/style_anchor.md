# 风格锚点

> 锚定写作风格，防止AI风格漂移。建库时从仿写分析中提取，后续每10章校准一次。

## 风格DNA

### 叙事视角
- **人称**：第三人称有限（跟随主角视角）
- **叙事距离**：贴近（读者能感受到主角的内心活动，但保留一定神秘感）
- **时态**：过去式

### 句式特征
- **基调**：轻松幽默为主，痞气主角自带笑点
- **节奏**：日常场景松弛、冲突场景紧凑、转折场景有张力
- **对话占比**：40-50%（主角嘴贱是核心特色）
- **描写密度**：重点场景不省笔墨（如大战、技术突破、情感爆发），日常场景简洁流畅

### 语言风格
- **用词倾向**：现代口语为主，技术/修炼描述时切换为精确书面语
- **幽默来源**：主角的毒舌吐槽 + 日常场景的荒诞感 + 配角与主角的反差
- **严肃场景处理**：该有的描写不省，但避免过度煽情；用细节和动作代替形容词堆砌
- **修辞密度**：中（幽默场景可适度增加，严肃场景收敛）

### 场景切换节奏
- 日常（慢档）：铺垫/羁绊/埋伏笔 → 每3-4章至少1章
- 冲突（中档）：次要矛盾升温未爆发 → 占60%以上
- 爆发（快档）：主线突破/大战/揭示 → 每卷2-3次，后必接慢/中档缓冲

### 对话风格
- 主角：痞气、毒舌、玩世不恭但话里有理
- 苏晚：冷淡寡言、开口必戳痛处
- 老哑巴：偶尔出口惊人
- 零番队成员：各有各的说话方式，不雷同

### 禁忌
- 禁止过度煽情（用行动代替抒情）
- 禁止主角说教（痞气角色不讲大道理）
- 禁止对话同质化（每个角色说话方式不同）
- 禁止AI痕迹（路标词、讲义腔、模板句式）
