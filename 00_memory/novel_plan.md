# 小说主线计划

> 本文件是长期记忆的核心，防止剧情偏离主线。每次写作前必读。
> 修改规则：仅在用户明确同意后修改。

## 核心设定（不可偏离）
- 书名：待定
- 类型：都市异能 + 科技修真 + 悬疑寻亲
- 核心卖点（一句话）：战后废墟中的11岁天才孤儿，凭借超高速大脑暗中掌握全人类知识，以科技解析修炼、建立全球势力寻找失踪父母，最终发现一切指向一个古老空间裂隙——而他的一生从出生起就被裂隙绑定了。
- 预计总字数：300章，每章不限长度
- 终极结局：主角在深境实验室地下20层解除空间裂隙封印 → 天魔涌出 → 真相揭露（被彼岸利用） → 零番队防守战悲壮牺牲24人 → 主角耗尽一切击溃裂隙后消失。尾章：25名幸存者散落各地等待集结暗号；雨夜街头疑似主角身影出现，竹签中传来微弱心跳。

  - 最终一击：所有力量 + 纳米大军 + 灵脉节点能量汇聚 → 轰入裂隙中心

## 核心人物弧线

### 主角（待命名）
- 起点（11岁）：表面痞气孤儿，暗中全知全能，伪装平凡
- 中期（14-15岁）：非洲基地建设者，零番队领袖，能力逐步暴露
- 后期（15-17岁）：复兴学院学员 + 修炼界异端 + 深境实验室渗透者
- 终点：发现一生都是裂隙棋局的一部分，选择牺牲自己关闭裂隙

### 苏晚（零番队001号）
- 起点：沉默寡言，密码天赋，怀表中的秘密
- 中期：零番队"影子指挥官"，主角唯一敢当面怼的人
- 终点：幸存25人之一，等待暗号重新集结

### 老哑巴（门扉001号）
- 表面：孤儿院扫地哑巴老人
- 真相：门扉计划第一代研究员，编号001，躲避彼岸追杀
- 关键作用：给主角能量不守恒笔记；在第三卷身份揭晓

### "镜"（彼岸联络人）
- 表面：调查父母失踪的热心公民
- 真相：逃逸天魔，整个帮助过程都是引导主角开门的棋局

### 林北
- 起点：学院死对头，军方背景
- 中期：家族与深境实验室合作曝光 → 被迫站队
- 终点：选择背叛家族站到主角一边

### 纪云书
- 起点：技术课教授，温和长者
- 真相：深境实验室内部线人，向外界传递实验室情报
- 关键作用：提供地下结构信息

## 绝对规则（红线）
1. 主角的"全知"只在暗中，任何时候不得在公开场合暴露真实能力（除非剧情需要逐步揭晓）
2. 修炼体系九境不可跨越大境界碾压，主角也必须逐境突破
3. 零番队49人的忠诚来源必须各不相同，不能用"洗脑""控制"等方式
4. 伏笔必须"前期埋、有间隔、后期收"，不得一章内埋收
5. 深境实验室、彼岸、净世会三方立场各自有逻辑，不得脸谱化
6. 结局主角消失暗示必须两可——读者可选择相信他还在
7. 空间裂隙的真相必须在第250章后才完全揭示，不得提前
